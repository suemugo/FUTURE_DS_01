# FUTURE_DS_01
Business Sales & Profit Performance Analysis

This project analyzes a Superstore sales dataset to evaluate business performance across categories, regions, and time.
The goal was to identify revenue drivers, profitability trends, and operational inefficiencies to provide actionable business recommendations.

1. Objectives

Analyze revenue and profit trends
Identify high-performing product categories
Evaluate regional profitability
Assess impact of discounting on profit
Develop a client-ready Power BI dashboard

2. Tools Used

Python (Pandas, Matplotlib)
Jupyter Notebook
Power BI Desktop
GitHub
